Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documents from a web server or from local storage and render the into multimedia web pages. HTML describes the structure of a web page semantically and originally includes cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML contructs, images and other objects, such as interactive forms, may be e mbedded into the rendered page. It provides a means to create structured douments by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other itmes. HTML elements are delineated by tags, written using angle brackets.
